## 本项目为手机销售系统项目开发
<br>

### 系统基本信息
<br>

##### 1. 分为`手机信息系统`和`员工信息系统`
##### 2. 有`员工（包括店长）`和`访客`两种访问权限（`访客无访问员工系统权限`）
<br>

### 系统基本介绍
##### 1. 手机信息系统 -- 面向访客
`1. 查询/打印手机信息以及购买手机等功能`
`2. 访客可以访问（访客登录无需账号即可访问登录账号为）`

##### 2. 手机信息系统 -- 面向员工
`1. 具备输入/修改/删除/查询手机信息等功能`
`2. 员工（店长）可以访问（登录账号为员工（店长）姓名的拼音（初始员工姓名拼音可参照“手机信息.txt”）初始密码为 123456）`

##### 3. 员工信息系统
`1. 具备输入/修改/删除/浏览人员信息等功能`
`2. 店长特有的权限，只有店长权限可以尝试登录访问（普通员工不可以）（登录账号为店长姓名的拼音（初始店长拼音姓名可参照“员工信息.txt”中职位为 manager 的员工信息）初始密码为 123456）`

##### 4. 注意事项
`1. 初始登录密码均为 123456`
`2. 各人员登录密码均可以更改`
